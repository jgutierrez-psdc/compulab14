Relational Fields Easy Search

The Main Idea:

This Module 'Easy Search' helps in searching in relational fields (Many2one,Many2many) by different fields pre defined in Configuration.

Key Features:

  • Search by different fields in relational fields
  • Automatic configuration for search fields

How it Works

Once Easy Search Installed, Take Access to Configuration menu by This Security Group.


Easy Search Config menu will appear.


Choose the module that you want to apply search on it
After This select searchable fields that you want yo used it
And you can change it at any time
(Notes: Non-stored field sale.order.display_name cannot be searched).


Enjoy Now by search
(search on customer by name,phone,mobile,state .... ). (search on product by internal ref,barcode,category .... ).


You Can Active Advance Search Feature in more model from general settings --> Easy Search Section
